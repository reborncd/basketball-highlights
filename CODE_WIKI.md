# 篮球进球集锦剪辑工具 - Code Wiki

## 1. 项目概览

篮球进球集锦剪辑工具是一个基于Python和PyQt6开发的应用程序，用于自动检测篮球比赛视频中的进球，并生成球员个人集锦。

### 主要功能：
- 自动检测篮球比赛视频中的进球
- 支持手动添加/删除进球时间戳
- 为每个进球分配球员
- 自动生成球员个人进球集锦
- 提供直观的图形用户界面

### 典型应用场景：
- 篮球比赛后快速生成球员个人集锦
- 野球场比赛记录与精彩瞬间提取
- 篮球训练分析与回顾

## 4. 核心模块

### 4.1 进球检测模块 (detector.py)

**功能**：自动检测视频中的篮球进球事件。

**核心类**：
- `DetectionConfig`：检测参数配置
- `BallTracker`：篮球检测与轨迹追踪
- `HoopDetector`：篮筐位置检测
- `GoalDetector`：综合判断是否进球

**工作原理**：
1. 检测篮筐位置
2. 追踪篮球轨迹
3. 分析篮球是否穿过篮筐
4. 检测球网是否有明显晃动
5. 综合判断是否进球

**关键函数**：
- `run_detection()`：对整个视频运行进球检测，返回进球时间戳列表
- `GoalDetector.process_frame()`：处理单帧，判断是否检测到进球
- `BallTracker.detect()`：检测篮球位置
- `HoopDetector.detect()`：检测篮筐位置

### 4.2 视频剪辑模块 (clipper.py)

**功能**：根据进球时间戳裁剪视频片段并合并成集锦。

**核心函数**：
- `clip_segment()`：从视频中截取指定时间段的片段
- `concat_clips()`：将多个视频片段合并成一个视频
- `get_video_info()`：获取视频基本信息
- `extract_thumbnail()`：从视频指定时间点提取缩略图

**技术实现**：
- 使用FFmpeg进行视频处理，支持快速stream copy（不重新编码）
- 支持生成视频缩略图用于预览

### 4.3 项目管理模块 (project.py)

**功能**：管理项目状态，包括进球时间戳、球员分配、片段路径等。

**核心类**：
- `Project`：项目管理类
- `GoalClip`：单个进球片段
- `Player`：球员信息

**关键功能**：
- 保存/加载项目状态
- 添加进球时间戳
- 管理球员信息
- 分配球员到进球片段
- 管理输出目录结构

## 5. GUI界面模块

### 5.1 主窗口 (main_window.py)

**功能**：应用程序的主窗口，包含三个标签页，分别对应检测、复核和导出功能。

**主要组件**：
- 菜单栏：新建项目、打开项目、退出
- 标签页：检测页面、复核页面、导出页面
- 状态栏：显示当前状态

**工作流程**：
1. 用户新建项目或打开现有项目
2. 切换到检测页面进行进球检测
3. 检测完成后切换到复核页面分配球员
4. 分配完成后切换到导出页面生成集锦

### 5.2 检测页面 (detection_page.py)

**功能**：导入视频并进行自动进球检测，支持手动添加/删除进球时间戳。

**主要组件**：
- 视频信息显示
- 检测参数设置
- 开始/停止检测按钮
- 进度条
- 进球片段列表
- 手动添加/删除按钮

**工作流程**：
1. 用户设置检测参数
2. 点击开始检测按钮
3. 程序后台运行检测算法
4. 显示检测进度
5. 检测完成后显示进球片段列表
6. 用户可以手动添加或删除进球时间戳

### 5.3 复核页面 (review_page.py)

**功能**：复核进球片段并为每个进球分配球员。

**主要组件**：
- 进球片段列表（含缩略图）
- 片段预览
- 球员按钮组
- 新建球员按钮
- 取消分配按钮

**工作流程**：
1. 程序自动生成每个进球的缩略图
2. 用户选择一个进球片段
3. 点击球员按钮为该片段分配球员
4. 重复上述步骤直到所有片段都分配了球员

### 5.4 导出页面 (export_page.py)

**功能**：裁剪视频片段并合并成球员个人集锦。

**主要组件**：
- 球员列表
- 输出目录显示
- 导出按钮
- 进度条

**工作流程**：
1. 用户选择要导出的球员
2. 点击导出按钮
3. 程序后台裁剪每个进球片段
4. 程序后台合并片段生成集锦
5. 显示导出进度
6. 导出完成后显示结果

## 6. 数据模型

### 6.1 GoalClip 类

**功能**：表示单个进球片段。

**属性**：
- `clip_id`：唯一ID
- `timestamp`：进球时间戳（秒）
- `start_sec`：片段开始时间
- `end_sec`：片段结束时间
- `player_name`：分配的球员名
- `clip_path`：已裁剪片段的文件路径
- `thumbnail_path`：缩略图路径
- `confidence`：置信度（"auto"或"manual"）
- `confirmed`：用户是否已确认

### 6.2 Player 类

**功能**：表示球员信息。

**属性**：
- `name`：球员名称
- `directory`：输出目录
- `highlight_path`：合并后集锦路径
- `color`：UI显示颜色

### 6.3 Project 类

**功能**：管理项目状态。

**属性**：
- `video_path`：视频文件路径
- `project_dir`：项目保存目录
- `clips`：进球片段列表
- `players`：球员列表
- `detection_done`：检测是否完成

**方法**：
- `save()`：保存项目状态
- `load()`：加载项目状态
- `add_goal()`：添加进球
- `add_player()`：添加球员
- `assign_player()`：分配球员到进球片段
- `get_clips_for_player()`：获取球员的进球片段
- `unassigned_clips()`：获取未分配的进球片段

## 8. 项目运行方式

### 8.1 安装依赖

```bash
pip install -r requirements.txt
```

### 8.2 安装FFmpeg

- **Windows**：下载FFmpeg并添加到系统PATH
- **macOS**：使用Homebrew安装 `brew install ffmpeg`
- **Linux**：使用包管理器安装 `sudo apt-get install ffmpeg`

```bash
python main.py
```

### 8.4 操作流程

1. **新建项目**：
   - 选择篮球录像文件
   - 选择项目保存目录

2. **进球检测**：
   - 设置检测参数（前滚时间、后滚时间、跳帧加速）

   - 等待检测完成

   - 查看每个进球的缩略图
   - 为每个进球分配球员
   - 可新建球员

4. **导出集锦**：
   - 选择要导出的球员
   - 点击"一键导出所有球员集锦"
   - 等待导出完成
   - 打开输出目录查看结果

## 9. 关键技术点

### 9.1 进球检测算法

**核心逻辑**：
1. **篮球检测**：使用HSV颜色空间检测橙色篮球
2. **轨迹追踪**：记录篮球运动轨迹
3. **篮筐检测**：使用Canny边缘检测和Hough圆变换检测篮筐
4. **穿越判断**：分析篮球是否从篮筐上方穿越到下方
5. **球网检测**：检测球网区域的像素变化（球网晃动）

**优化措施**：
- 跳帧处理：每N帧处理一次，提高检测速度
- 冷却时间：防止短时间内重复检测同一进球
- 多信号融合：轨迹信号和球网信号结合，提高检测准确率

### 9.2 视频处理技术

**核心技术**：
- 使用FFmpeg进行快速视频剪辑（stream copy）
- 使用FFmpeg合并多个视频片段
- 使用FFmpeg提取视频缩略图
- 多线程处理：后台运行检测和导出任务，不阻塞UI

### 9.3 GUI设计

**设计特点**：
- 三页式标签布局：检测 → 复核 → 导出
- 响应式界面：适应不同窗口大小
- 实时进度显示：检测和导出过程中显示进度
- 直观的视觉反馈：使用颜色和图标表示状态
- 后台任务处理：避免UI卡顿

## 10. 配置与参数

### 10.1 检测参数

| 参数 | 默认值 | 说明 |
|------|-------|------|
| pre_roll | 5.0 | 进球前保留秒数 |
| post_roll | 3.0 | 进球后保留秒数 |
| min_ball_radius | 8 | 篮球最小半径(像素) |
| max_ball_radius | 60 | 篮球最大半径(像素) |

| net_change_threshold | 0.03 | 球网变化阈值 |
| trajectory_min_frames | 5 | 判断轨迹所需最少帧数 |
| goal_cooldown | 8.0 | 两次进球最短间隔(秒) |
| sample_every_n | 2 | 每N帧处理一次，加快速度 |

## 11. 故障排除

### 11.1 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 无法打开视频 | 视频格式不支持 | 尝试转换为MP4格式 |
| 检测不到进球 | 篮球颜色与背景相似 | 调整篮球检测参数 |
| 篮筐检测失败 | 篮筐在画面中不明显 | 尝试调整篮筐检测参数 |
| 导出失败 | FFmpeg未安装 | 安装FFmpeg并添加到PATH |
| 程序崩溃 | 内存不足 | 处理较短的视频或增加系统内存 |

### 11.2 日志信息

程序运行时会生成日志信息，可用于排查问题：
- 检测过程中的日志：显示检测进度和进球时间戳
- 导出过程中的日志：显示裁剪和合并进度
- 错误信息：显示具体的错误原因
